# Power BI HR Analytics Dashboard

This project is a complete and interactive **HR Analytics Dashboard built using Power BI**. It helps in analyzing employee-related data to gain meaningful business insights for Human Resource decision-making. The dashboard is designed with a clean UI, dynamic slicers, and automated filtering using bookmarks and buttons.

This project can be used as a **portfolio project for Data Analyst, Power BI Developer, and HR Analytics roles**.

---

## Project Objective

The main objective of this project is to:
- Analyze employee hiring trends over different months
- Understand employee attrition behavior
- Visualize salary group distributions
- Analyze overtime work patterns
- Provide interactive filtering using slicers for better decision-making

This dashboard allows HR teams and management to make **data-driven decisions** based on employee performance, workload, and retention patterns.

## Data Model Overview

The dashboard follows a **star-schema-based data model** with:
- A central **Employee** fact table
- Supporting dimension tables such as:
  - EducationLevel
  - PerformanceRating
  - RatingLevel
  - SatisfiedLevel

This structure ensures:
- Accurate filtering
- Better performance
- Clean relationship management
